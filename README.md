# College Virtual Assistant

A React + TypeScript based AI-powered college assistant built using Gemini API.

## Features
- AI chatbot
- Face scanner
- Dashboard
- ...

## Tech Stack
- React
- TypeScript
- Vite
- Gemini API

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
